# m-home-lab

## 项目简介

`m-home-lab` 是一个用于归档和发布多个独立 HTML 页面的仓库。这里存放着一些零散的页面资源，可能包含工具页面、信息展示页、实验性界面等，每个页面均为独立 HTML 文件，不依赖后端服务，可直接在浏览器中运行。

本仓库不针对任何特定域名或网站，仅作为个人页面集合的版本管理与发布平台。

## 目录结构

```
m-home-lab/
├── index.html          # 仓库入口页（可选）
├── pages/              # 存放所有独立 HTML 页面
│   ├── page-a.html
│   ├── page-b.html
│   └── ...
├── assets/             # 公共资源（CSS、JS、图片等）
│   ├── css/
│   ├── js/
│   └── images/
└── README.md           # 本文件
```

- `pages/`：所有独立 HTML 页面均存放于此，每个文件为一个独立页面。
- `assets/`：存放页面间共享的静态资源文件，如样式表、脚本、图片等。

## 页面归档说明

- 每个页面均为自包含的 HTML 文件，可直接通过浏览器打开运行。
- 页面之间无硬性依赖，可独立访问。
- 页面内容可能包含实验性功能或临时信息，不保证长期可用或完整。
- 部分页面可能引用 `assets/` 目录下的公共资源，请确保文件路径正确。

## 维护说明

- 本仓库持续更新，不定期新增或修改页面。
- 页面内容可能随时调整，不提供版本回退承诺，但 Git 历史记录可供参考。
- 若发现页面无法正常显示，请检查文件路径或资源引用是否完整。
- 欢迎通过 Issues 提交问题或建议，但不保证及时响应。

## 使用方式

1. 克隆本仓库到本地：
   ```bash
   git clone https://github.com/your-username/m-home-lab.git
   ```
2. 直接在浏览器中打开 `pages/` 目录下的任意 HTML 文件即可查看对应页面。
3. 也可通过 GitHub Pages 或其他静态托管服务部署后在线访问（需自行配置）。
